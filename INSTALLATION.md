#  Installation Guide

##  Download & Run  

1. **Download the latest release**  
   - Go to [Releases](https://github.com/olfa-ga/sensor-data-emulator-cli-dashboard/releases/latest)  
   - Download **`dashboard.exe`**  

2. **Run the Program**  
   - Double-click `dashboard.exe` to start  
   - Press **'q'** to exit
